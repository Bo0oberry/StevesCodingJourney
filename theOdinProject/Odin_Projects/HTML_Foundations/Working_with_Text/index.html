How do you create a heading in HTML?

use h1 - h6 tags

How many different levels of headings are there and what is the difference between them?

there are 6 headings h1 - h6 with h1 being the most important and havingthe largest font size and h6 being the least important and having the smallest font size

What element should you use to make text bold and important?

Use the strong tag. It also marks the text as important, this affects tools like screen readers where the voice tone may change to indicate importance

What element should you use to make text italicized to add emphasis to it?

Use the em tag. It works similary to strong but instead of bold the text is italicized

What relationship does an element have with any nested elements within it?

The element nested inside another element is called a child element and the element it's inside is called the parent. Some css stlyings appied to the parent element, will be inherited by the child

What relationship do two elements have if they are at the same level of nesting?

Two, or more elements that are at the same level of nesting are called sibling elements
