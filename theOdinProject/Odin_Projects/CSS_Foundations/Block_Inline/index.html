What is the difference between a block element and an inline element?

  • Inline:

    • Height and Width are only affected by content
    • Top and botttom padding will overlap with other elements

  • Block

    • Height and Width values are respected
    • element is set on new line
    • padding will push other elements away
What is the difference between an inline element and an inline-block element?

The inline-block:

  • Height & Width values are respected
  • top / bottom padding will push elements awaay
  • will NOT start on new line, instead will be set inline with prev element
